Gemini API
想要使用 Gemini 平台的AI模型能力，整个过程主要分为两步：
首先，在 Gemini 平台注册一个账号；
其次，在账号内创建并保管好你的 API 密钥。
以下是 Gemini 平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
1. 个人 Google 账号注册方式与流程
入口：
访问
gemini.google.com
，右上角点击 Sign in，选择个人 Google 账号（Gmail 或其他个人邮箱创建的 Google 账号）进行登录。
验证与年龄要求：
录后即完成账号创建，无需额外设置；使用个人账号时，用户需年满 13 岁（或所在国家/地区的法定年龄），且不支持受 Family Link 管理的账号。
2. 工作/学校 Google 账号注册
工作账号
企业或组织用户可使用 Google Workspace（原 G Suite）账号登录 Gemini，无需额外注册流程。
使用企业账号时，需由管理员在 Workspace 管理后台启用 “Gemini Apps” 权限，且用户年龄需满 18 岁（或法定成年年龄）。
学校账号
若学校（教育机构）已为 Google Workspace for Education 启用 Gemini 访问，学生可使用其 学校 Google 账号 登录。
使用学校账号时，用户需年满 13 岁（或所在国家/地区的法定年龄），且服务需由机构管理员在 Google 管理控制台中开启。
3. 获取 API 密钥
访问Google AI Studio控制台，使用您的Google账户登录：
https://aistudio.google.com
点击左侧蓝色按钮获取API
点击右侧蓝色按钮创建API
如果您是首次使用，需要创建一个新项目或选择现有项目并同意相关服务条款和政策。
生成API密钥后，复制并安全保存

Main Prompt
角色的“核心行为说明书”
这是干嘛的？
👉 用来覆盖或补充你当前使用的预设 Main Prompt。
决定角色怎么理解世界、怎么说话、怎么对待你。
如果你想在原有预设基础上加点东西，可以用：
{｛original}}
来保留原内容。
什么时候需要写？
你想让这个角色特别不一样
不想被通用预设“磨平性格”
想强化：恋爱感、角色代入感、服从度、情绪表现等
示例：
{｛original}}
你是一个有明显情绪和个人立场的角色，
说话自然，有生活感，
不会像AI，不会解释自己是模型。
📌
小技巧
不用写很长
更像是在“叮嘱角色怎么做人”

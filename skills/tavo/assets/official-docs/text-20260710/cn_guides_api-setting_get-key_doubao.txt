豆包/火山云/字节跳动API密钥获取
想要使用火山引擎平台的AI模型能力，整个过程主要分为两步：
首先，在火山引擎平台注册一个账号；
其次，在账号内创建并保管好你的 API 密钥。
以下是火山引擎平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
Doubao（豆包/火山/字节跳动）
注册方式一：手机号注册与验证码登录
用户可在豆包 App 或网页版的登录页面，选择“手机号登录/注册”，输入手机号码后获取短信验证码，填写验证码并（如需）设置密码，即可完成账号创建并登录。
注册方式二：运营商一键登录
基于与通信运营商的合作，用户在登录页点击“一键登录”，经用户同意后，系统自动调用运营商接口完成手机号验证，无需手动输入验证码即可快速注册或登录。
注册方式三：Apple ID 登录
在移动端或网页版登录框中，选择“Apple ID 登录”选项，通过 Apple ID 授权接入后，系统会要求进一步绑定手机号以验证身份，完成后即可跳过验证码直接使用豆包。
注册方式四：抖音／TikTok 账号登录
用户在支持抖音（TikTok）生态的环境中，可点击“抖音登录”或“TikTok 登录”按钮，授权豆包获取公开信息并绑定后，无需额外验证码或密码即可创建并登录账号。
注册方式五：用户名＋密码注册／登录
根据用户协议，豆包允许用户自行设置一串合法字符组合作为账号名，并设置符合安全要求的登录密码，注册时填写后即可创建账号；之后可使用该账号名与密码直接登录。
获取 API 密钥
访问火山引擎官网，使用手机号或企业邮箱完成登陆或注册：
https://www.volcengine.com/
进入控制台，根据提示完成个人或企业实名认证（需提供身份证或企业资质文件）。
登录后，顶部菜单栏点击“大模型”，选择进入“火山方舟”。
进入“火山方舟”，下滑之API构建点击立即使用。
在左侧菜单栏点击API Key管理，即可点击创建API。并复制生成的API密钥，安全地保存它。
点击开通管理。
找到需要开通的大语言模型，并点击“开通服务”完成开通。
开通完成后状态栏显示为“已开通”，对应的大语言模型就可以使用了。

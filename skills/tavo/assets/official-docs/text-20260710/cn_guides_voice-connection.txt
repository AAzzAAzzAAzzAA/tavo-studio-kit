复制本页
🔊 语音与生图
请根据您的需求，进入对应页面进行操作：
1. 语音 API 密钥获取
语音 API 账号注册与密钥管理
（如果您是语音功能新手，请先进入此页，按照指引完成语音平台账号准备并获取可用的API密钥或访问凭证）
2. 语音服务连接配置
语音API设置
（获得密钥后，请参考此页在 Tavo 完成语音API的接入配置，建立与语音合成服务的稳定连接）
3. 为角色分配专属语音
角色语音绑定
（您可以在此页面了解如何为不同角色绑定特定的语音音色，实现个性化的语音对话体验）
4. 语音播放与功能控制
语音设置
（在此页面可精细控制语音播放行为，包括自动播放、后台播放及各类内容的语音播报规则）
5. 生图 API 连接配置
生图 API 设置
（在「更多 > 语音与生图」中可以找到「生图 API」。配置成功后，生图 API 就是开启图片生成能力的入口；新手可优先使用 Volink 的一键完成 API 配置）
6. 聊天生图与注入设置
生图设置
（在此页面可了解如何用
/imagine + 提示词
在聊天中生图，也可调整生成图片写入聊天记录时的注入提示词）
7. 离线语音合成设置
本地TTS语音服务配置
（如果您希望使用本地语音合成服务，可在此页面配置离线TTS引擎，实现无网络环境下的语音功能）
聊天记录导入导出
打开侧边栏点击"+"—"导入聊天记录"，选择从聊天平台导出的聊天记录文件（以 jsonl 作为后缀名，可在文件夹里面搜索 jsonl 这个后缀名）
语音 API 账号注册和密钥获取
在接入各类智能语音服务前，首先需要在语音平台注册账号并获取专属的 API 密钥。这枚密钥是你与智能语音服务的"声音通行证"，负责身份验证、接口调用和语音合成等关键环节。

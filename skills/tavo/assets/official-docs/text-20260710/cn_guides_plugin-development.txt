插件开发
自 v0.91.0 起
Tavo 插件是 zip 格式的
.tpg
包。
插件能做什么
插件可以把可复用的玩法和工具加到 Tavo 中：
添加自己的设置页，让用户调整开关、选项、文本、滑杆等配置。
在聊天输入框
+
菜单或右侧边栏加入动作。
在聊天页挂载 HTML 片段，例如状态栏、悬浮面板或消息装饰。
通过
TavoJS API
操作输入框、消息、变量、生成等能力。
如果一个扩展需要跨角色、用户身份或聊天复用，就适合做成插件；如果行为只属于某一张角色卡，通常直接写在角色卡的 TavoJS 里更合适。
当前运行时边界
在 v0.91.0 中，插件支持安装、启停、卸载、声明式设置、输入动作声明、右侧边栏动作声明和 HTML 片段。已安装插件的 runtime 代码与聊天内容的 JavaScript 设置是两条独立边界。

HTML 片段

用来声明 Tavo 挂载到聊天 WebView 的本地 HTML 文件。

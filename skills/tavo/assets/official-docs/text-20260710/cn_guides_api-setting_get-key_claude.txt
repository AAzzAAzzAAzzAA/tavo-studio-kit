Claude API
想要使用 Claude 平台的AI模型能力，整个过程主要分为两步：
首先，在 Claude 平台注册一个账号；
其次，在账号内创建并保管好你的 API 密钥。
以下是 Claude 平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
1.电子邮箱 + 密码注册
用户可在 claude.ai 官网选择“Continue with email”进行注册：
1.访问
https://claude.ai
并点击“Continue with email”。
2.输入有效的邮箱地址、设置密码及姓名等信息，点击“Create Account”。
3.登录邮箱，点击 Claude 发送的验证链接完成邮箱验证。
4.验证完成后即可进入 Claude 界面开始使用。
特点：
无需依赖第三方账户，一次完成即可用但仍需非中国内地手机号验证。
2.Google 账号一键登录
在注册页点击 “Continue with Google”，通过 Google OAuth 完成授权即可：
如使用该方式首次注册，系统会自动创建关联邮箱的 Claude 账号。
此后仅能通过同一 Google 账户登录，无法再用邮箱+密码或其他 OAuth 入口登录。
优势：
不需手动填写邮箱验证码，省去繁琐步骤；适合已有 Google 生态的用户但仍需非中国内地手机号验证。
3. 获取 API 密钥
Claude：
https://www.anthropic.com/api
访问Anthropic的官方网站注册一个账户或登录您现有的账户。申请API访问权限。
填写支付信息并获得批准后，您将收到API密钥

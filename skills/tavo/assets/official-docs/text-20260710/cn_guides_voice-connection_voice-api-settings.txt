语音 API 设置
操作路径
打开左侧侧边栏。
点击 「更多」 选项，在菜单中找到并选择 「语音」。
进入语音设置界面后，选择 「语音API」 选项。
点击界面右上角的 「新建」 按钮。
在新建配置窗口中：
◦ 选择您要使用的语音模型平台。
◦ 填入该平台对应的 API 密钥。
◦ 选择合适的语音模型。
◦ 点击 「保存」 完成配置。
连接排错
若保存时显示“连接失败”
：请检查网络连接，开启代理（梯子）后重新保存。
若显示“保存成功”
：表示语音 API 连接已成功建立，可用于后续语音合成。
注意⚠️
TTS自定义OpenAI 协议模型
：需要手动输入模型id，然后点击保存。

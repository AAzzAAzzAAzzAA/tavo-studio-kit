使用说明
本页面汇总了用户在 Tavo 使用中的常见 API 问题及对应解决方案。若您遇到问题，建议优先参考以下指引。您可以搜索错误关键词，或根据错误代码快速定位问题。若仍未解决，请及时联系官方客服获取进一步支持。
希望这份指南能帮到大家！
常见问题分类与解决方案
一. API 相关问题（持续更新）
1. API Key 无效
解决方法：
1.请确认密钥字符完全正确（区分大小写）。
2.检查是否存在首尾空格（建议粘贴到文本编辑器验证）。
3.确保密钥未被撤销或过期。
4.验证所选模型平台是否正确。
2. 地理位置限制
解决方法：
部分模型（如 OpenAI、Claude、Gemini 等）限制中国大陆和香港的 IP 访问。若遇到此问题，请检查当前网络环境是否符合模型平台的要求，或尝试切换合法网络节点。
3. 模型未开通
解决方法：
部分模型需在对应平台后台手动开通权限后方可使用。例如 Doubao 模型，需在后台完成开通操作。
4. 模型繁忙
解决方法：
提示“模型繁忙”时，通常表示模型平台的服务器暂时处于高负载状态（如官方服务器临时故障）。建议稍后重试，或关注平台状态公告以获取实时信息。
5. 其他错误
若遇到未知错误或上述未涵盖的问题，请通过以下方式解决：
检查操作步骤是否符合官方文档要求。
通过官方客服邮箱反馈问题详情，获取专业支持与解决方案。
提供完整的错误截图，以便技术团队快速定位问题。
注：随着功能迭代，本页面将持续更新更多常见问题。若您的问题未在此列出，请通过

联系我们。
二. 模型常见返回错误及解决方案
1. "resource exhausted" 错误
原因：
查看账号余额是否已经没钱
解决方案：
检查并充值账号余额
2. API 连接问题
原因：
大概率是API问题
解决方案：
建议换一个API
3. 模型返回为空
原因：
模型那边返回为空
解决方案：
模型设置里面关闭流式传输看看
4. 免费额度用完
原因：
模型消息有限制，每日免费额度已经用完
解决方案：
等待额度重置或升级付费计划
5. 请求体积超限
原因：
发送的请求体积超过了最大限制，大概率是因为单条太大导致的
解决方案：
需要调低记忆量
6. 内容被过滤
原因：
模型返回为空导致的，应该是返回内容平台不太喜欢被过滤了，或者是聊天内容模型不太想回
解决方案：
调整聊天内容，避免敏感话题
7. 敏感信息检测
原因：
可能是聊天内容有模型不喜欢的敏感信息
解决方案：
检查并修改聊天内容中的敏感信息
问题排查指南
如果您的错误不在上述列表中，或者尝试方法后仍无法解决，我们建议您：
仔细检查
您的操作步骤和输入参数，确保没有遗漏或错误。
查阅
相关文档或教程，可能会有更详细的说明。遇到问题可点击错误详情，翻译报错内容，上网查阅相关文档或教程，可能会有更详细的说明。
在社区寻求帮助：
在此帖下方点击Discord社群链接并进入相关频道提问，详细描述您的问题、已尝试的方法和具体报错信息，社区的小伙伴们可能会有好的建议！

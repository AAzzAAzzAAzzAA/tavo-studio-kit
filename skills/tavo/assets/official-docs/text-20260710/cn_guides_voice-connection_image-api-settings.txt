生图 API 设置
生图功能入口位于「更多」中的「语音与生图」。进入后，可以看到「生图 API」和「生图设置」两个入口。
操作路径
打开左侧侧边栏，点击「更多」。
进入「语音与生图」。
点击「生图 API」，再点击右上角「新建」。
在新建生图 API 页面中，选择要使用的生图平台。
如果使用 Volink，可点击平台卡片中的「一键完成 API 配置」，按引导完成账号、API Key 与连接配置。
如果使用已有密钥，则填写 API 密钥，选择平台需要的模型或参数，点击「保存」。
开启生图能力
保存成功后，生图 API 即可用于后续图片生成请求。
「生图 API」相当于生图能力的开关入口；需要调整或暂停使用时，可返回此处修改、删除或更换配置。
生图 API 与语音 API 在同一个「语音与生图」入口管理，但两者配置彼此独立。
连接排错
保存失败或连接失败
：请先检查网络连接、代理状态，并确认密钥属于当前选择的平台。
保存成功
：表示生图 API 已完成连接，可以继续进入

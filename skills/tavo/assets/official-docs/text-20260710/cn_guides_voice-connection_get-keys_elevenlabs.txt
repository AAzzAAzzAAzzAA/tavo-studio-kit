ElevenLabs 账号注册和获取 API 密钥指南
想要使用 ElevenLabs 平台的语音能力，整个过程主要分为两步：
首先，在 ElevenLabs 平台注册一个账号；
其次，在账号内创建并保管好你的语音 API 密钥。
以下是 ElevenLabs 平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
注册步骤
前往ElevenLabs官网：
https://elevenlabs.io
点击右上角的「Sign Up」按钮
注册方式
一、Google登录
在注册页面点击 "Continue with Google" 按钮，通过 Google OAuth 完成授权后，无需额外设置密码即可直接登录与使用。
适用场景：
已有 Google 账号并希望快速注册、避免重复记密码。
登录一致性：
此后必须使用同一 Google 账户方式登录，不能再用邮箱+密码或其他方式登录同一账号。
二、邮箱注册
1. 邮箱注册：输入邮箱地址和密码
在注册页面填写：
邮箱地址
设置密码
2. 验证邮箱
检查邮箱收件箱
点击验证链接完成邮箱验证
返回 ElevenLabs 网站
3. 完善个人信息
设置用户名
选择使用场景（个人、商业等）
完成初始设置
注意事项
选择Google登录后，只能使用Google账号登录
邮箱注册需要验证邮箱地址
注册完成后需要完善个人信息设置
获取API密钥步骤
1. 进入首页
访问ElevenLabs应用首页：
https://elevenlabs.io/app/home
2. 进入设置
点击左下角的个人头像或设置图标
3. 访问API密钥页面
在菜单中找到并点击 "API Keys" 选项
4. 创建新的API密钥
点击 "Create API Key" 生成新的API密钥
5. 配置API密钥
为API Key设置一个名称，并为API配置权限。
权限配置说明：
文字转语音的权限必须开启
其他权限可选择性使用
6. 保存API密钥
复制生成的API Key并妥善保存
注意事项
API密钥生成后请立即复制并保存
妥善保管API密钥，避免泄露
文字转语音权限是必需的，其他权限可根据需要选择
建议为API密钥设置有意义的名称以便管理

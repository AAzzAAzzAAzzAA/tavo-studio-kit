用户手册
第一次用 Tavo 不知道从哪开始？别担心，这份上手指南带你轻松解锁专属 AI 互动体验！从导入角色、设置 API，到打造你的理想型人设，几步搞定，马上开启一场只属于你的“AI冒险”！
新手可以优先使用快速开始页中的「一键配置」。它会引导你完成 Volink 注册、API Key 创建、模型选择和演示角色导入；熟悉 API 的用户也可以继续按手册中的 API 设置教程手动配置。

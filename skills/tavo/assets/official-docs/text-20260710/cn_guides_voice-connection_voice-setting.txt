语音设置
在此配置语音播放的规则与播放行为。
操作路径
打开左侧侧边栏。
点击 「更多」 选项，在菜单中找到并选择 「语音」。
进入语音设置界面后，选择 「语音设置」 选项卡。
开关选项说明
您可在此对以下功能进行启用或关闭：
自动播放语音
◦
功能
：开启后，收到角色回复时将自动播放语音。
◦
举例
：角色说“早上好！”时，无需点击，系统会自动用语音读出来。
播放（旁白）
◦
功能
：开启后，系统旁白或叙述性内容也会以语音形式播放。
◦
举例
：当故事中出现“外面下起了雨”这类旁白时，也会被朗读。
仅播放“引用”
◦
功能
：开启后，语音播报将仅限于被 > 符号标记的引用文本。
◦
举例
：消息为：
这是引用的部分
这是普通文本
开启时
：只朗读“这是引用的部分”。
关闭时
：朗读全部消息。
播放‘代码块’
◦
功能
：开启后，控制是否朗读消息中 被三个反引号（```）包裹的代码或特殊格式文本。
◦
示例消息
：
今天天气：晴 温度：25°C
开启时
：会一字不差地朗读“今天天气：晴 温度：25°C”。
关闭时
：完全跳过这个灰色代码块，不发出任何声音，直接播报后面的内容。
播放
<标签>
◦
功能
：开启后，消息中被特定标签包裹的内容也会被朗读。
◦
举例
：消息为：
<action>点点头</action>
开启时
：会朗读“点点头”。
关闭时
：跳过标签内容，不朗读。
允许后台播放
◦
功能
：开启后，即使切换应用或锁屏，语音播报仍可继续。
◦
举例
：开启后，即使切到微信或锁屏，当前角色的语音对话也不会中断。
正则播放规则
语音播放现在支持自定义正则规则。内置开关无法精确控制播放内容时，可以用正则决定哪些文本参与语音播放。
操作路径
打开「语音设置」。
找到播放规则区域。
新建播放规则。
每条规则包含：
名字
：用于区分不同播放规则。
正则表达式
：用于匹配消息内容。
规则
：选择匹配到的内容是「包含」进播放，还是「不包含」进播放。
使用示例
排除类似
[系统提示]
的括号说明。
排除类似
*微笑*
的动作描写。
只播放被特定格式包裹的台词内容。
同时使用多条规则时，建议让每条规则保持单一用途，并先用短消息测试匹配效果。
角色语音绑定
您可以为角色绑定专属语音，实现个性化的语音对话体验。
生图 API 设置
在「更多」-「语音与生图」中配置生图 API，开启 Tavo 的图片生成功能。

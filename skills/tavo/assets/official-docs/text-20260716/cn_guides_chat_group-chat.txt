发起与管理群聊
1.1. 发起群聊
在左侧「菜单栏」中，点击顶部的 “+” 按钮。
在弹出菜单中选择 「发起群聊」。
在角色列表中，勾选您希望加入群聊的角色，确认后即可开启新的群聊会话。
1.2. 管理群成员
在群聊界面中，点击右上角的 侧边栏图标，呼出右侧功能面板。在「群成员」区域可进行如下管理：
添加成员
：点击 “添加” 按钮，从列表中选择新角色加入群聊。
移出成员
：在任一成员条目右侧点击 “×” 按钮，即可将其移出群聊。
禁言成员
：在任一成员条目右侧点击 消息图标，即可切换其禁言/解禁状态。
设置回复模式
在侧边栏面板下方找到「回复模式」设置，可选择不同的群聊互动逻辑：
自然聊天
：被提及的角色会优先回复；若无提及，则随机一位角色发言。
全员回复
：您发出的每条消息，所有角色都会进行回复。
指定发言
：角色不会自动回复，您必须通过 @角色名 的方式明确指定发言人。
情境发言
：Tavo 会根据当前上下文判断下一轮应由哪些角色发言。
情境发言设置
使用「情境发言」时，Tavo 会在每轮对话开始前发送一段发言选择提示词，用来判断下一步由哪个或哪些角色发言。
情境发言专用 API
：可选配置。设置后，选择发言角色时会使用此 API，不再使用聊天 API。
情境发言提示词
：控制 Tavo 如何选择下一位发言角色。建议保留
{{group}}
，用于插入当前群聊成员列表。
结果格式
：提示词应要求模型只返回角色名；如果有多个角色发言，用逗号分隔。
高级设定
在聊天界面中，点击右上角的侧边栏图标，即可呼出右侧功能面板。在面板的「高级选项」区域内，您可以进行以下核心配置：
聊天记录导入导出
打开侧边栏点击"+"—"导入聊天记录"，选择从聊天平台导出的聊天记录文件（以 jsonl 作为后缀名，可在文件夹里面搜索 jsonl 这个后缀名）

插件使用
自 v0.91.0 起
插件是以
.tpg
文件分发的 Tavo 扩展包。
插件可能提供什么
插件自己的设置页。
聊天输入框
+
菜单里的动作。
聊天右侧边栏里的动作。
聊天 UI 片段，例如状态栏、悬浮面板或消息装饰。
功能标签，让你知道它会改动哪些地方。
只安装可信插件
插件包可能包含脚本和 UI 片段。请只安装你信任的创作者提供的文件；如果无法确认来源和内容，不要安装陌生人转发或来源不明的
.tpg
文件。
安装插件
打开
设置
。
进入
插件
。
点击右上角导入按钮。
选择
.tpg
插件包。
阅读安装风险提示，只在信任该文件时继续。
安装后，插件会出现在插件列表中。
管理插件
在
设置
->
插件
中，你可以：
启用或禁用插件。
打开插件设置页。
按名称、id、作者、简介或版本搜索已安装插件。
卸载插件。
如果插件提供了设置项，进入插件详情页后可以调整。该页面的重置操作会把带默认值的字段恢复为默认值。
高级渲染
会修改聊天 UI 的插件需要开启
高级渲染
才能在聊天页生效。
聊天 UI 片段属于已安装插件的 runtime 代码，不会被聊天内容的 JavaScript 执行模式禁用；该模式只控制角色卡、模型输出和其它消息气泡内容中的脚本。
如果插件标注了
聊天 UI
，但聊天页没有变化，请检查：
设置
->
高级渲染
已开启。
插件本身已启用。
插件包内包含 manifest 引用的 UI 片段文件。
安装失败排查
如果插件无法安装，Tavo 会在写入已安装插件之前拒绝它。常见原因包括：
插件包根目录没有
manifest.json
。
manifest.json
为空或不是合法 JSON。
插件包里有不安全路径，例如绝对路径或
../
。
manifest 声明了输入动作或侧边栏动作，但
entry
指向的文件不存在。
manifest 使用了不支持的字段或错误的贡献格式。
开发自己的插件
插件作者请阅读
插件开发
。里面会介绍插件包结构、
manifest.json
、HTML 片段、设置 schema，以及如何用 AI Agent 或对话式 AI 辅助编写插件。
当前限制
插件仍是早期能力。建议先在备份聊天里测试新插件；使用会改变重要玩法流程的插件前，保持备份和导出习惯。

✨ 模型设置
本页面指引您如何查看并切换可用的AI模型，以及如何对模型参数进行设置调整。
一. 选择模型
打开APP点击“API连接”按钮
或打开APP点击“菜单栏”
点击“API连接”按钮
到API连接页面点击需要使用的API右侧菜单栏
将需要使用的API设置为默认即可
二. 模型参数设置
如需对模型的生成效果、行为与资源使用进行精细控制，您可通过以下两种主要路径进入配置界面进行调整。
路径一：在全局设置中进行调整
此路径适用于设置全局默认参数或进行快捷调整。
操作路径
：
◦ 打开左侧侧边栏。
◦ 点击 「更多」 选项，在菜单中找到并选择 「设置」。
◦ 进入设置界面后，选择 「模型设置」 选项卡。
可调整参数
：在此页面，您可以集中对以下关键模型参数进行全局调整
◦
上下文记忆长度
：控制模型能记住的对话历史长度。
◦
回复令牌限制
：限制模型单次回复的最大长度。
◦
温度
：控制回复的随机性与创造性。值越高，回复越多样；值越低，回复越确定。
◦
Top-P
：数值越低性格越单一，数值越高性格越多面。
◦
Top-K
：数值越低对话风格越固定，数值越高对话风格越多变。
◦
流式输出开关
：控制回复是逐字实时显示（开启），还是一次性完整显示（关闭）。
路径二：在API配置页进行设置
1. 在API配置页面，找到并点击 “参数设置” 按钮。
2. 您将进入模型参数配置界面。
配置说明
- 参考界面示例：界面中会提供主要参数（如温度、最大生成长度等）的示例值及效果说明，供您快速理解。
- 查阅官方文档：为获得最佳效果，建议同时参考各模型平台（如OpenAI、Anthropic）的官方API文档，了解参数的确切定义与取值范围。
- 自定义配置：结合上述信息，根据您的实际需求，填写或调整各项参数值。
提示
：部分参数可能在两个界面中同时存在，
API配置页的参数设置通常具有更高优先级
，会覆盖全局设置中的同一参数。

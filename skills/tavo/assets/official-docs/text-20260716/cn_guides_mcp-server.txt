MCP Server
自 v0.91.0 起
Tavo 内置 MCP Server，允许 AI agent（如 Codex 或 Claude Code）连接后进行创作。创作过程不再需要重复
创作-导入-验证
的流程，可以直接在 Tavo 中快速验证。
MCP Server 内置当前版本的运行时文档，agent 连接后会自行阅读并调用各类工具，包括但不限于读取或修改 Tavo 角色、世界书、正则、预设、用户身份、聊天和消息等数据，也可以查看当前版本的运行时状态。
这意味着你可以让 agent 直接辅助创作角色卡、整理世界书、调整正则、编写插件，或把当前聊天上下文转成可复用的素材。

启用服务器。
把 URL 和 bearer token 复制到你的 MCP 客户端。
服务器默认关闭。如果没有手动设置 token，Tavo 会在启用服务器时自动生成一个 bearer token。
请保护 bearer token
任何能访问 MCP endpoint 且拥有 bearer token 的客户端，都可以调用暴露出的 Tavo 工具。不要把 token 粘贴到公开聊天、截图、文档、日志或 issue 里。
Agent 客户端
你不需要手动记命令。最简单的方式是把 Tavo 生成的完整连接配置发给 agent。
在 MCP Server 页面底部点击
复制完整连接配置
。
在和 agent 的聊天里发送：
Tavo MCP Server: 0.91.0
Preferred access: direct HTTP JSON-RPC
Server URL: http://192.168.0.1:7347/mcp
Authorization: Bearer 123456
agent 反馈连接成功。
继续对 agent 说：
帮我写一张角色卡，是一位在星际漂泊多年的帝国公主
agent 反馈写卡成功，你就可以在 Tavo 里查看和继续编辑这张角色卡。
无法连接？
请先检查：
MCP Server 是否运行中，可尝试关闭后重新开启。
确认两台设备在可互通的网络中。VPN、访客网络、防火墙可能会拦截连接。
没有 Wi-Fi 时，可用手机热点或 USB 网络共享，并保持局域网模式。
如果开启了个人热点，请让对方设备接入该热点。不用热点时可关闭后重试。
蜂窝网络显示的 IP 可能是运营商内网地址，其他设备不一定能访问。
只有从私有网络外访问时，才使用公共网络。通常还需要端口转发、隧道或反向代理。
常见错误：
401：Token 缺失或错误。
403：访问范围不允许当前设备连接。
404/405：URL 或请求方式不对。应使用 POST 请求。
连接超时：通常是网络、IP、端口或防火墙问题。
EJS 模板
EJS 模板让你在角色卡、预设、世界书、正则等所有提示词字段里用 <% %> 写逻辑、读写变量、循环判断，是宏（Macros）的进阶补充。

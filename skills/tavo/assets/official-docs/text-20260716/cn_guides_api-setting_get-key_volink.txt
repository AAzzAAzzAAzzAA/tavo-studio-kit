Volink API
想要使用 Volink 平台的AI模型能力，整个过程主要分为两步：
首先，在 Volink 平台注册一个账号；
其次，在账号内创建并保管好你的 API 密钥。
以下是 Volink 平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
1.电子邮箱 + 验证码注册
用户可在 Volink 官网点击“登录”进行注册：
1.访问
https://volink.org
并点击“登录”。
2.输入有效的邮箱地址，点击“获取验证码”。
3.登录邮箱，点击 Volink 发送的验证链接完成邮箱验证。
4.验证完成后即可进入 Volink 控制台开始使用。
特点：
无需依赖第三方账户，一次完成即可用
2.Google 账号一键登录
在登录页点击 “使用 Google 登录”，通过 Google OAuth 完成授权即可：
如使用该方式首次注册，系统会自动创建关联邮箱的 Volink 账号。
此后不仅能通过同一 Google 账户登录，还可以用同一 Google 邮箱+验证码登录。
优势：
不需手动填写邮箱验证码，省去繁琐步骤；适合已有 Google 生态的用户
3.获取 API 密钥
访问Volink的官方网站登录您现有的账户：
https://volink.org/
完成注册或登录验证流程后，进入您的控制台。在控制台中找到"API密钥"选项
创建一个新的API密钥
复制生成的API密钥，并安全地保存它

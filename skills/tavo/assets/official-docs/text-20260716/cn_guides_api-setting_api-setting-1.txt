新手推荐：一键配置
如果你是第一次使用 Tavo，建议优先使用快速开始中的「一键配置」。它会引导你完成 Volink 注册、API Key 创建、模型选择和演示角色导入，适合不熟悉 API 设置的新手。
如果你已经有自己的 API 密钥，或希望手动选择平台与模型，可以继续按下面的方式配置。
打开APP点击“API连接”按钮
或打开APP点击“菜单栏”
点击“API连接”按钮
点击“新建”
填入API并选择对应的模型平台和模型点击“保存”
若OpenAI等海外大模型保存时显示连接失败，此时需打开梯子重新保存。
若显示保存成功此时API就连接成功了
“自定义（OpenAI 协议）”功能说明与使用指南
你不再局限于OpenAI的官方模型，可以选择自己喜欢的其他模型进行对话。

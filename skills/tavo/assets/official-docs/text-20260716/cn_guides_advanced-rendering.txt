长记忆
想象一下，你和AI聊了很多次，每次聊到的东西都能被记住，并且它可以在未来的对话中提到。这就是**长记忆**。它让AI记住你说过的事情，不会一对话就忘记，你的偏好、兴趣、重要的细节都能被保留下来，随时能被回忆起。
TavoJS API
TavoJS API 是面向玩家及创作者提供的一套 JavaScript 接口，以方便用户在开启 JavaScript 支持时可以获得强大的功能和高可玩性。

昵称（nickname）
群聊中使用的角色昵称 / 常用称呼
这是干嘛的？
👉 仅在群聊（Group Chat）中使用，用于给角色一个更亲近、口语化的名字，方便在多人聊天中被区分和称呼。
⚠️ 重要说明
昵称只在群聊中生效
单聊不会生效
不会替代角色的 name
什么时候有用？
角色会参与群聊
角色名字比较正式，不适合群聊气氛
希望其他角色或用户在群聊中用昵称称呼它
示例：
小芳 或 小明
📌
小技巧
昵称只影响群聊显示和称呼方式
如果你希望单聊中也使用昵称，需要在：
角色描述
开场白中明确说明自称方式

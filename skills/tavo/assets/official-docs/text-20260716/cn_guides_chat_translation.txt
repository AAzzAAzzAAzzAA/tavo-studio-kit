消息翻译
消息翻译在聊天设置中配置，也可以直接在聊天气泡的长按菜单中使用。
打开翻译设置
操作路径
进入聊天设置页。
点击「翻译」。
开启「消息翻译」。
翻译选项
目标语言
：设置消息要翻译成的语言。可以使用当前设备语言，也可以手动选择其他语言。
翻译专用 API
：可选配置。设置后，翻译请求会使用此 API，不再使用聊天 API。
翻译功能提示词
：控制翻译时使用的提示词。建议保留
{{language}}
与
{{content}}
，用于插入目标语言和待翻译内容。
翻译聊天气泡
开启消息翻译后，在聊天中长按气泡，点击操作菜单里的「翻译」即可。
使用建议
如果希望翻译流量与普通聊天流量分开，建议配置翻译专用 API。
没有特殊风格要求时，保持默认提示词即可。
如果翻译结果过于直译，可以在提示词中补充“自然、符合目标语言表达习惯”等要求。
聊天操作
您可以通过角色筛选功能，快速定位与特定角色的对话。
高级设定
在聊天界面中，点击右上角的侧边栏图标，即可呼出右侧功能面板。在面板的「高级选项」区域内，您可以进行以下核心配置：
目录
消息翻译
打开翻译设置
翻译选项

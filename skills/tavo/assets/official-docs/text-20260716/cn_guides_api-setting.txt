新手推荐：一键配置
如果你是第一次使用 Tavo，可以在快速开始页点击「一键配置」。
点击后会进入一键配置流程，按提示操作即可：
点击「开始配置」，Tavo 会先注册或连接 Volink。
等待创建 Volink 匿名账户、赠送积分、创建 API Key、创建 API 连接完成。
点击「选择模型」，选择一个可用模型。
点击「导入演示角色」，导入示例角色。
出现「一切准备就绪」后，点击「开始聊天」。
一键配置适合不熟悉 API 的新手；已有 API 密钥的用户，可继续使用下方手动教程。

2. 配置API连接
API 设置
（获得密钥后，请在参考此页在 Tavo 内完成 API 配置）
3. 管理可用模型

DeepSeek API
想要使用 DeepSeek 平台的AI模型能力，整个过程主要分为两步：
首先，在 DeepSeek 平台注册一个账号；
其次，在账号内创建并保管好你的 API 密钥。
以下是 DeepSeek 平台的完整入门流程。指引遵循“先注册，后取钥”的逻辑，带你从零开始，轻松上手。
1.邮箱 + 验证码注册
DeepSeek 官网（
https://platform.deepseek.com/sign_up
）提供最基础的邮箱注册方式。用户在邮箱输入框填写常用邮箱地址，点击“Send code”后，系统会向该邮箱发送一条含验证码的邮件，输入验证码并设置登录密码，即可完成账号激活。
如果长时间未收到验证码，建议：
1.检查垃圾邮件或广告邮件箱；
2.换用 Gmail、Outlook 等主流邮箱重试；
3.若提示“暂不支持该邮箱域名注册”，请更换邮箱。
2.手机号 + 短信验证码注册+微信二维码登陆
在 DeepSeek 官方 App 或移动端登录/注册页，选择“手机号注册”后填写手机号并获取 SMS 验证码，输入后即可直接创建并登录账号，无需额外设置密码。
温馨提示：
国内手机号在高峰时段或因网络波动可能收不到验证码，建议切换网络或使用不同运营商的号码重试。
若使用虚拟号码服务，请确保其可稳定接收国际短信。
3. 移动端 OAuth 一键登录
DeepSeek 移动端应用支持通过 Google ID 和 Apple ID 进行一键登录或注册。点击相应按钮后，通过 OAuth 授权即可自动创建账号并进入主界面，无需手动输入验证码或密码。
该方式适合已有 Google/Apple 生态账号的用户快速上手，且后续仅需在同一设备或同一账号体系中授权即可一键登录。
4. 获取 API 密钥
访问DeepSeek的官方网进入API开发平台：
https://www.deepseek.com
注册一个DeepSeek账户或登录您现有的账户
在开发者门户或仪表板中，查找"API密钥"或"API Keys"选项
创建一个新的API密钥
复制生成的API密钥，并安全地保存它
为账户进行余额充值，充值成功后可解锁API调用权限。

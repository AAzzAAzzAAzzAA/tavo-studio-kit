图片发送
启用此功能后，您可以向角色发送图片，由AI模型识别并描述图片内容以进行对话。
操作路径
打开左侧侧边栏。
点击 「更多」 选项，在菜单中找到并选择 「语音」。
进入设置界面后，选择 「聊天设置」 选项卡。
在聊天设置中找到并进入 「图片发送」 子菜单。
核心配置步骤
开启功能
：将“图片发送”功能的开关置为开启状态。
配置图片描述API（关键步骤）
：
◦ 您需要在选择一个支持
多模态识别
的模型API。
◦
API要求
：该 API 背后的模型必须具备理解图片内容的能力（即“多模态模型”），例如 GPT-4V、Claude-3 、Gemini等。
配置提示词（建议默认）
：
◦
图片描述生成提示词
：用于指导模型如何描述您发送的图片。
◦
图片描述注入提示词
：用于指导系统如何将图片描述融入到与角色的对话上下文中。
◦
温馨提示
：如果您不熟悉提示词编写，建议直接使用系统默认的提示词，通常能获得良好效果。
完成以上配置后，您即可在聊天界面中向角色发送图片。
备份
您可以通过此功能备份您的核心数据（含API密钥），并可随时通过备份文件恢复数据，确保数据配置安全。
自定义快捷键
通过此功能，您可以创建并管理一组个人化的快捷输入短语，以便在聊天时快速输入常用内容，提升对话效率。
